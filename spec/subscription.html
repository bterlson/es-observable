CreateSubscription ( _observer_ )

The abstract operation CreateSubscription with argument _observer_ is used to create a Subscription object. It performs the following steps:

1. Assert: Type(_observer_) is Object. 1. Let _subscription_ be ObjectCreate(%SubscriptionPrototype%, «‍ [[Observer]], [[Cleanup]] »). 1. Set _subscription_.[[Observer]] to _observer_. 1. Set _subscription_.[[Cleanup]] to *undefined*. 1. Return _subscription_.

Set _subscription_.[[Cleanup]] to *undefined*. 1. Let _result_ be Call(_cleanup_, *undefined*, «‍ »). 1. WarnIfAbrupt(_result_, *undefined*). 1. Return *undefined*.

SubscriptionClosed ( _subscription_ )

The abstract operation SubscriptionClosed with argument _subscription_ performs the following steps:

1. Assert: _subscription_ is a Subscription object. 1. If _subscription_.[[Observer]] is *undefined*, return *true*. 1. Else, return *false*.
